chooseImage(OBJECT)

从相册选图或使用相机拍照。

OBJECT参数说明：

参数 类型 必填 说明
sourceType String 否 album 从相册选图，camera 使用相机，无数据/其他的时候 二者都允许
success Function 是 接口调用成功的回调函数

返回参数说明：

参数 类型 说明
data String 图片Base64字符串
code Number 调用反馈标识，成功：0 ；失败：-1
msg String 调用反馈信息

示例代码：

tqb.chooseImage({
  success: function (res) {
    // 返回选定照片的本地文件路径列表，tempFilePath可以作为img标签的src属性显示图片
    var tempFilePaths = res.tempFilePaths
  }
})

getLocation(OBJECT)

获取当前的地理位置。当用户离开淘钱宝后，此接口无法调用；

OBJECT参数说明：

参数 类型 必填 说明
success Function 是 接口调用成功的回调函数，返回内容详见返回参数说明。

success返回参数说明：

参数 类型 说明
data Object 返回数据对象
data.latitude Number latitude：纬度；均为浮点数
data.longitude Number longitude：经度，均为浮点数
data.address String 返回当前的地理位置对应实际地址
data.addressDetail Object 返回实际地址对象 province:省；city:市；region:区；street:街道
code Number 调用反馈标识，成功：0 ；失败：-1
msg String 调用失败反馈信息

示例代码：

tqb.getLocation({
  success: function(res) {
    var latitude = res.latitude
    var longitude = res.longitude
    var speed = res.speed
    var accuracy = res.accuracy
  }
})

navigateBack(OBJECT)

关闭当前页面，返回上一页面或多级页面。

OBJECT 参数说明：

参数 类型 默认值 说明
delta Number 1 返回的页面数，如果 delta 大于现有页面数，则返回到首页。

success返回参数说明：

参数 类型 说明
code Number 调用反馈标识，成功：0 ；失败：-1
msg String 调用失败反馈信息

示例代码：



// 此处是A页面
tqb.navigateTo({
  url: 'B?id=1'
})

// 此处是B页面
tqb.navigateTo({
  url: 'C?id=1'
})

// 在C页面内 navigateBack，将返回A页面
tqb.navigateBack({
  delta: 2
})
